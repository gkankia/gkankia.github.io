About the project

Population Lines

The visualization shows population density in Georgian context using the number of people living within a 1 sq.km grid. Inspired by James Cheshire's work.

Areas coloured in red are Russian-occupied territories with no data available

DATA: Geostat 2014 General Population Census. TOOLS USED: qgis & qgis2three plugin
